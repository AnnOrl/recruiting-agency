# Подбор персонала

Для хранения сессии используется Redus
### Локальный Redus

Установка

`wget http://download.redis.io/releases/redis-6.0.3.tar.gz`

`tar xzf redis-6.0.3.tar.gz`

`cd redis-6.0.3`

`make`

Запуск

`src/redis-server`

### Запуск приложения

`npm i`

`npm npm run build` 

`npm start`
